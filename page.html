Semantic HTML Tags

Understanding Semantic Tags

Semantic tags clearly define the meaning of the content. They help both the browser and the developer understand the purpose of the content.

Quick Tips

  • Use <header> for page header
  • Use <nav> for navigation
  • Use <main> for main content
  • Use <footer> for page footer
